# Smart Energy & Environment Monitor

## Release 1 – Core Prototype

This project demonstrates a basic IoT pipeline using simulated sensors.

### What works in Release 1
- Simulated temperature sensor using Cisco Packet Tracer
- Python script running on the sensor
- UDP communication to a PC
- PC-side listener receives and logs data
- Regular data generation
- Simple processing (message counting)

### Architecture
Packet Tracer Sensor → UDP → PC Listener

### Technologies Used
- Cisco Packet Tracer (IoT simulation)
- Python
- UDP sockets
- JSON formatted messages

### Status
Release 1 complete.
